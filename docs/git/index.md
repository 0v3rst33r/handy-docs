# git commands

## Create branch

```
git checkout -b BRANCH_NAME
```

## Delete branch

### Delete (safe) branch

```
git branch -d BRANCH_NAME
```

### Delete (force) branch

```
git branch -D BRANCH_NAME
```

## Fetch - with pruning

```
git fetch --prune
```

## List branches

### List local branches

```
git branch
```

### List remote branches

```
git branch -r
```
